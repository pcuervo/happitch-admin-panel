This service creates a new PitchResult

POST /pitch_results

Parameters:

  • auth_token* (string): Authorization token generated after for each user after login.
  • pitch_id* (integer)
  • agency_id* (integer)
  • was_proposal_presented (boolean)
  • got_response (boolean)
  • was_pitch_won (boolean)
  • got_feedback (boolean)
  • when_will_you_get_response (date)
  • when_are_you_presenting (date)
  • codeExample params
    {
    "auth_token": "J6f5ax8C8Y9GyCBkgx83",
    "pitch_result": {
        "agency_id": 1,
        "pitch_id": 2,
        "was_proposal_presented": false,
        "got_response": false,
        "was_pitch_won": false,
        "got_feedback": false,
        "has_someone_else_won": false,
        "when_will_you_get_response": "2016-10-17",
        "when_are_you_presenting": "2016-10-17"
    }
}
  • doneSuccessful Response

    HTTP 201 Created. Returns the newly created PitchResult object

    {
  "id": 1,
  "agency_id": 1,

  "was_proposal_presented": false,
  "got_response": false,
  "was_pitch_won": false,

  "has_someone_else_won": false,

  "when_are_you_presenting": "2016-10-17",
  "created_at": "2016-10-19T21:53:58.380-05:00",
  "updated_at": "2016-10-19T21:53:58.380-05:00"
}
  • clearError Responses

    HTTP 422 Unprocessable Entity. When Agency is empty or missing

    {
  "errors": {
    "agency": [
      "La agencia es obligatoria"
    ]
  }
}

    HTTP 422 Unprocessable Entity. When Pitch is empty or not found

    {
  "errors": {
    "pitch": [
      "El pitch es obligatorio"
    ]
  }
}
