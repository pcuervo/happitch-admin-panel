This service is used to search by keyword the active pitches of an agency.

POST /pitch_evaluations/search

Parameters:

  • auth_token* (string): Authorization token generated after for each user after login.
  • keyword* (string): The keyword to look for in the Pitche's name.
  • codeExample params
    {   
    "keyword": "est26",
    "auth_token": "bLnbTvCsNAn4eDFBB2aZ"
}
