This service updates a PitchEvaluation

POST /pitch_evaluations/update

Parameters:

  • auth_token* (string): Authorization token generated after for each user after login.
  • id* (integer) - PitchEvaluation id
  • pitch_id* (integer)
  • are_objectives_clear* (boolean)
  • has_selection_criteria* (boolean)
  • time_to_present* (string): Can be '1s', '2s', '3s' or '4s'
  • is_budget_known* (boolean)
  • number_of_agencies* (string): Can be '2 - 4', '>4' or 'NA'
  • are_deliverables_clear* (boolean)
  • is_marketing_involved* (string): Can be 'si', 'no', or 'no se'
  • time_to_know_decision* (string): Can be '2s', '3s', '4s', '5s', '>6' or 'NA'
  • deliver_copyright_for_pitching* (string): Can be 'si', 'no', or 'no se'
  • number_of_rounds* (string): Can be '1r', '2r', '3r' or '4r'
