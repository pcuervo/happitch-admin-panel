This service creates a new PitchWinnerSurvey

POST /pitch_winner_surveys

Parameters:

  • auth_token* (string): Authorization token generated after for each user after login.
  • pitch_id* (integer)
  • agency_id* (integer)
  • was_contract_signed* (boolean)
  • was_project_activated* (boolean)
  • contract_signature_date* (date)
  • when_will_it_activate* (date)
