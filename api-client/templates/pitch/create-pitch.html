This service creates a new Pitch

POST /pitches

Parameters:

  • auth_token* (string): Authorization token generated after for each user after login.
  • name* (string)
  • brand_id* (integer)
  • brief_date* (date)
  • brief_email_contact* (string)
  • skill_categories* (array)
