This service is used to filter pitches of an agency.

POST /pitch_evaluations/search

Parameters:

  • auth_token* (string): Authorization token generated after for each user after login.
  • happitch (boolean)
  • happy (boolean)
  • ok (boolean)
  • unhappy (boolean)
  • archived (boolean)
  • cancelled (boolean)
  • declined (boolean)
  • codeExample params
    {   
    "auth_token": "QZHUoyjhFWLijzRmP847",
    "happitch": true,
    "archived": true
}
  • doneSuccessful Response

    HTTP 200 OK. Returns the PitchEvaluations' details.
