This service deletes an existing success case

POST /success_casers/destroy

Parameters:

  • id* (integer): ID of existing agency
  • auth_token* (string): Authorization token generated after for each user after login.
  • codeExample params
    {   
    "id": 4,
    "auth_token": "5uKB2vzUY1vApmkQ9Vg_"
}
  • doneSuccessful Response

    HTTP 204 No Content. No JSON response

  • clearError Responses

    HTTP 422 Unprocessable Entity. When there is no success case found

    {
  "errors": "No se encontró ningún case de éxito con id: 4"
}
Response:

{{ successCaseResponse | json }}
