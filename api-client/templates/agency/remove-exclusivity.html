This service removes exclusive brands from an agency

POST /agencies/remove_exclusivity_brands

Parameters:

  • auth_token* (string): Authorization token generated after for each user after login.
  • id* (integer) - Agency ID
  • brands* (array) - An array of brands' ids.
  • codeExample params
    {
    "auth_token": "kCEg9ryafBVy2n6Ps7qR",
    "id": "157",
    "brands": ["44"]
}
  • doneSuccessful Response

    HTTP 201 Created. Returns the exclusivity brands array from the agency

    [
  {
    "id": 61,
    "agency_id": 219,
    "brand": "Pepsi",
    "created_at": "2016-11-01T14:04:53.419-06:00",
    "updated_at": "2016-11-01T14:04:53.419-06:00"
  }
]
  • clearError Responses

    HTTP 422 Unprocessable Entity. When agency is not found

    {
  "errors": "No se encontró la agencia con id: "
}

    HTTP 422 Unprocessable Entity. When there are no brands sent

    {
  "errors": "No se encontró ninguna marca de exclusividad"
}
