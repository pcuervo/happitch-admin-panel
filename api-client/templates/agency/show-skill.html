This service fetches an existing Skill

GET /skills/:id

Parameters:

  • id (integer)
  • doneSuccessful Response

    HTTP 200 OK. Returns a Skill object

    {
  "id": 1,
  "name": "Aplicaciones móvil",
  "skill_category": {
    "id": 3,
    "name": "Desarrollo",
    "created_at": null,
    "updated_at": null
  }
}
  • clearError Responses

    HTTP 422 Unprocessable Entity. When there is no Skill found

    {
  "errors": "No existe ningún skill con id: 12"
}
Response:

{{ skillCatResponse | json }}
