This service adds skills to an agency

POST /agencies/add_skills

Parameters:

  • auth_token* (string): Authorization token generated after for each user after login.
  • id* (integer) - Agency ID
  • skills* (array) - An array of skills with id and level.
  • codeExample params
    {   
    "auth_token": "5uKB2vzUY1vApmkQ9Vg_",
    "id": 1,
    "skills" : [
        {
            "id": 1,
            "level": 3
        },
        {
            "id": 2,
            "level": 4
        }
    ]
}
