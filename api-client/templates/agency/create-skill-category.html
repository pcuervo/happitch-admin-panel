This service creates a new SkillCategory

POST /skill_categories

Parameters:

  • auth_token* (string): Authorization token generated after for each user after login.
  • name* (string)
  • codeExample params
    {   
    "auth_token": "5uKB2vzUY1vApmkQ9Vg_",
    "skill_category" : {
        "name": "Publicidad"
    }
}
  • doneSuccessful Response

    HTTP 201 Created. Returns the newly created SkillCategory object

    {
  "id": 1,
  "name": "Publicidad",
  "skills": []
}
  • clearError Responses

    HTTP 422 Unprocessable Entity. When SkillCategory name alreadys exists

    {
  "errors": {
    "name": [
      "Ya existe una categoría con ese nombre"
    ]
  }
}

    HTTP 422 Unprocessable Entity. When name is empty or missing

    {
  "errors": {
    "name": [
      "El nombre de la categoría no puede estar vacío"
    ]
  }
}
Response:

{{ skillCatResponse | json }}
