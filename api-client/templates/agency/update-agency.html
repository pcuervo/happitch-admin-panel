This service updates an existing agency

POST /agencies/update

Parameters:

  • id* (integer): ID of existing agency
  • auth_token* (string): Authorization token generated after for each user after login.
  • name* (string)
  • phone (string)
  • contact_name (string)
  • contact_email (string): Valid email address
  • address (string)
  • latitude (decimal)
  • longitude (decimal)
  • website_url (string)
  • num_employees (integer)
  • golden_pitch (boolean)
  • silver_pitch (boolean)
  • high_risk_pitch (boolean)
  • medium_risk_pitch (boolean)
  • logo ( base64 image )
  • codeExample params
