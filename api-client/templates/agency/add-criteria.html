This service adds criteria to an agency

POST /agencies/add_criteria

Parameters:

  • auth_token* (string): Authorization token generated after for each user after login.
  • id* (integer) - Agency ID
  • criteria* (array) - An array of criteria ids.
  • codeExample params
    {   
    "auth_token": "ZEatomyf8K9qXsVLxyE4",
    "id": 1,
    "criteria": [ 1, 2, 3 ]
}
