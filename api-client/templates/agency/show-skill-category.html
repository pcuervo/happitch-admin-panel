This service fetches an existing SkillCategory

GET /skill_categories/:id

Parameters:

  • id (integer)
  • doneSuccessful Response

    HTTP 200 OK. Returns an agency object

    {
  "id": 1,
  "name": "Publicidad",
  "skills": []
}
  • clearError Responses

    HTTP 422 Unprocessable Entity. When there is no SkillCategory found

    {
  "errors": "No se encontró ninguna categoría de skills con id: 123"
}
Response:

{{ skillCatResponse | json }}
