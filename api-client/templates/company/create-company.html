This service creates a new company

POST /companies

Parameters:

  • auth_token* (string): Authorization token generated after for each user after login.
  • name* (string)
  • codeExample params
    {   
    "auth_token": "t2838qq6W9evS8X2snFX",
    "company": 
    {
        "name": "The Coca Cola Company"
    }
}
  • doneSuccessful Response

    HTTP 201 Created. Returns the newly created Company object

    {
  "id": 4,
  "name": "The Coca Cola Company",
  "brands": []
}
  • clearError Responses

    HTTP 422 Unprocessable Entity. When there is no Company name present

    {
  "errors": {
    "name": [
      "El nombre de la compañía no puede estar vacío"
    ]
  }
}
Response:

{{ companyResponse | json }}
