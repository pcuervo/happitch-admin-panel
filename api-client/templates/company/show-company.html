This service fetches an existing Company

GET /companies/:id

Parameters:

  • id (integer)
  • doneSuccessful Response

    HTTP 200 OK. Returns a Company object

    {
  "id": 1,
  "name": "Anderson, Streich and Keebler",
  "brands": []
}
  • clearError Responses

    HTTP 422 Unprocessable Entity. When there is no Company found

    {
  "errors": "No se encontró la compañía con id: 11"
}
Response:

{{ companyResponse | json }}
