This service updates an existing company

POST /companies/update

Parameters:

  • auth_token* (string): Authorization token generated after for each user after login.
  • id* (integer) - Company ID
  • name (string)
  • contact_name (string)
  • contact_email (string)
  • contact_position (string)
  • logo (base64)
  • codeExample params
    {   
    "auth_token": "t2838qq6W9evS8X2snFX",
    "id": 181,
    "filename": "img.png",
    "logo": "BASE64",
    "company": 
    {
        "name": "The Coca Cola Company",
        "contact_name": "Juan Pérez",
        "contact_email": "juan@cocacola.com",
        "contact_position": "Brand Manager"
    }
}
  • doneSuccessful Response

    HTTP 201 Created. Returns the newly created Company object

    {
  "id": 4,
  "name": "The Coca Cola Company",
  "contact_name": "Juan Pérez",
  "contact_email": "juan@cocacoal.com",
  "contact_position": "Brand Manager",
  "brands": ['Sprite','Squirt']
}
  • clearError Responses

    HTTP 422 Unprocessable Entity. When there is no Company name present

    {
  "errors": {
    "name": [
      "El nombre de la compañía no puede estar vacío"
    ]
  }
}
