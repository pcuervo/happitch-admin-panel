This service creates a new Brand

POST /brands

Parameters:

  • auth_token* (string): Authorization token generated after for each user after login.
  • name* (string)
  • contact_name (string)
  • contact_email (string)
  • contact_position (string)
  • company_id* (id)
  • codeExample params
    {   
    "auth_token": "t2838qq6W9evS8X2snFX",
    "brand": 
    {
        "name": "Pepsi",
        "contact_name": "Pepe Pecas",
        "contact_email": "pepe@pepsi.com",
        "contact_position": "Brand Manager",
        "company_id": 5
    }
}
  • doneSuccessful Response

    HTTP 201 Created. Returns the newly created Brand object

    {
  "id": 3,
  "name": "Pepsi",
  "contact_name": "Pepe Pecas",
  "contact_email": "pepe@pepsi.com",
  "contact_position": "Brand Manager",
  "company": {
    "id": 5,
    "name": "PepsiCo"
  }
}
  • clearError Responses

    HTTP 422 Unprocessable Entity. When name is empty or missing

    {
  "errors": {
    "name": [
      "El nombre de la marca no puede estar vacío"
    ]
  }
}

    HTTP 422 Unprocessable Entity. When company is empty or not found

    {
  "errors": {
    "company": [
      "La compañía es obligatoria"
    ]
  }
}
Response:

{{ companyResponse | json }}
