This service return the dashboard summary of a user.

POST /pitch_evaluations/dashboard_summary_by_user

Parameters:

  • auth_token* (string): Authorization token generated after for each user after login.
  • id* (integer): User ID
  • codeExample params
    {   
    "id":1,
    "auth_token": "teqECHs82jW3KFojexp-"
}
  • doneSuccessful Response

    HTTP 200 OK. Returns the number of pitches by type as well as won and lost pitches for a given user.

    {
  "happitch": 1,
  "happy": 0,
  "ok": 1,
  "unhappy": 1,
  "lost": 0,
  "won": 0
}
  • clearError Responses

    HTTP 422 Unprocessable Entity. When there is no user found

    {
  "errors": "No user found with id: 10101"
}
