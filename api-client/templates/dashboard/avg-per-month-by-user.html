This service return the average score of pitches per month, by user.

POST /pitch_evaluations/average_per_month_by_user

Parameters:

  • auth_token* (string): Authorization token generated after for each user after login.
  • codeExample params
    {   
    "id":1,
    "auth_token": "teqECHs82jW3KFojexp-"
}
  • doneSuccessful Response

    HTTP 200 OK. Returns the scores grouped by month.
