This service return the average score of pitches per month, by agency.

POST /pitch_evaluations/average_per_month_by_agency

Parameters:

  • auth_token* (string): Authorization token generated after for each user after login.
  • id* (integer): Agency ID
  • codeExample params
    {   
    "id":1,
    "auth_token": "teqECHs82jW3KFojexp-"
}
